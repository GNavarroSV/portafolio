Proyectos.

Estos son algunos de los proyectos que he desarrollado, haciendo uso de diferentes tecnologias y cada uno de ellos orientado a diferentes propósitos.

TekedaFash

Tienda 'Ecommerce' para un nuevo emprendimiento de la moda salvadoreño. Permitiendo a los usuarios ver el catálogo de productos, registrarse y realizar un pedido. Adicionalmente permite a los administradores realizar las gestiones correspondientes de usuarios, compras, ventas y reportes.

Soluciones Digitales

Sistema contable que permite a la empresa 'Soluciones Digitales' realizar el registro de todas las transacciones que son realizadas tomando en cuenta el catálogo de cuentas correspondiente y generando los asientos contables, estados y balances financieros necesarios. De igual forma, permite registrar proyecciones de sus servicios bajo el método de órdenes de trabajo.

Sistema para la Gestión de Capturas

Sistema orientado al registro y gestión de capturas realizadas por la policia nacional civil. El sistema permite realizar el registro del capturado y sus datos personales junto a los cargos que se le atribuyen y lugar de detención. Adicionalmente, el sistema permite registrar y llevar control de los traslados y procesos judiciales que enfrente el detenido.
